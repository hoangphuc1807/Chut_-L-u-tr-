# E-Commerce Mobile App

> Platform mua sắm di động với giao diện hiện đại và thanh toán an toàn.

## 📱 Features Chính

### Phase 1 (MVP)
- [ ] User registration & login
- [ ] Product catalog
- [ ] Search & filter
- [ ] Shopping cart
- [ ] Checkout process
- [ ] Order tracking

### Phase 2 (Future)
- [ ] Push notifications
- [ ] Reviews & ratings
- [ ] Wishlist
- [ ] Loyalty program

## 🏗️ Architecture

```
Frontend (React Native)
    ↓
API Gateway (Node.js)
    ↓
Backend Services
    ├── User Service
    ├── Product Service
    ├── Order Service
    └── Payment Service
    ↓
Database (PostgreSQL)
```

## 🚀 Cách Chạy

```bash
# Install dependencies
npm install

# Run development server
npm start

# Run tests
npm test
```
