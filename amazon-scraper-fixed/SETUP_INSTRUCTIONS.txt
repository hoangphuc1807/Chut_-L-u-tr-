🚀 FIRST USE GUIDE
─────────────────────────────────────────────────────────────────────────────

   1. Click extension icon (toolbar, top right)
   2. Enter keywords (one per line):
      - gifts for mom
      - gifts for dad
      - gifts for wife
   3. Select pages: "3 trang (full)" or "1 trang" for quick test
   4. Click: 🚀 Bắt đầu cào dữ liệu
   5. Watch progress with 4 stat boxes
   6. Excel auto-downloads when done
   7. Click 📊 Dashboard to see history

📊 STATS BOX EXPLAINED
─────────────────────────────────────────────────────────────────────────────

   ┌──────────┬──────────┬──────────┬──────────┐
   │    0     │    0     │    0     │   0/0    │
   │ Cần      │ Hợp lệ  │ Loại bỏ │ Keywords │
   │ Verify   │   ✅    │   ❌    │ Progress │
   └──────────┴──────────┴──────────┴──────────┘

   • Cần Verify: Total products found
   • Hợp lệ ✅: Products available (passed filter)
   • Loại bỏ ❌: Products skipped (unavailable)
   • Keywords Progress: How many keywords done (x/total)

🛠️ TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────

   Problem: Extension doesn't appear
   Solution: Refresh chrome://extensions (F5)

   Problem: Popup is blank
   Solution: Right-click > Remove, then reload unpacked

   Problem: Scraping is slow
   Solution: Check internet, close other tabs

   Problem: Excel doesn't download
   Solution: Check Downloads folder, try Dashboard > Export

   Problem: Dashboard shows no sessions
   Solution: Refresh page (F5), check scraping completed
