❓ QUICK FAQ:

Q: Is the extension broken?
A: No! It still works exactly like before. This is just an addition.

Q: Do I have to use Supabase?
A: No. If you don't configure it, extension works normally (Excel only).

Q: Does this cost money?
A: Supabase free tier is plenty (500MB storage). Metabase is free.

Q: Can I disable this feature?
A: Yes. Just don't enter Supabase URL/Key. Extension will skip cloud sync.

Q: What if I mess up?
A: No problem. Start fresh - create new Supabase project.

Q: How long does setup take?
A: ~30 minutes first time. Then it's automatic.

Q: Can multiple people scrape?
A: Yes! All data goes to same Supabase.

Q: Will my Excel export still work?
A: Yes! Both methods work (Excel + Supabase).
